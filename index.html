SPT Planning Calendar Builder Guide

Fill in the panel on the left and the schedule builds itself in the preview. Everything below explains what each control does and how the calendar is calculated. Nothing you enter leaves your computer — it all runs in your browser.

Show Info

Title / Season names the project. It appears in the printed Excel header and becomes the file name when you export or save (e.g. “Show Title - Season 1 Planning Calendar.xlsx”).

Shooting Days per Episode is informational context for the shoot — it’s shown in the header’s production summary. The actual shoot length is driven by the total shooting-day count you set on the Production phase.

Calendar range

Optional. By default the calendar spans the full year(s) your schedule touches (January–December) so exports are a consistent size. Set a start and/or end date here to force a different window — useful if you want extra lead-in or trailing weeks.

Phases

Each phase is a block of the schedule — Writer’s Room, Pre Prep, Prod Prep, Production, and Post are provided by default, each with its own color. For every phase you set:

  • Start date — any date snaps to the Monday of that week (weeks always run Monday–Sunday).
  • Duration — in weeks for most phases. Production is special: it’s measured in shooting days, not weeks, and is scheduled day-by-day (see Union holidays below).
  • Name — click the phase name to rename it; the label carries through to every week of that phase and to the export.

+ Add phase creates a custom phase with its own name, color, start, and length. Phases can overlap — when two or more run in the same week, the preview and export add columns so they sit side by side. Columns follow a clear rule: the earliest-starting phase takes the leftmost column and keeps it, later overlapping phases sit to its right, and non-overlapping phases share a column to keep the sheet narrow. When only one phase is running in a week, it spans across the empty columns so there are no blank gaps.

Header (Auto vs. Manual). By default the calendar header auto-fills from your inputs (title, production span, Principal Photography dates, episode count) and updates live. Click Header: Auto — Switch to Manual above the calendar to take control: it snapshots the current values into editable lines you can freely edit, and stops auto-updating. Switch back to Auto to discard the manual edits and resume live fill.

Union holidays

Applies to the Production phase only. Pick a country and, if a recognized union holiday lands on what would have been a shoot day, that day is skipped and the shoot extends day-by-day until the full shooting-day count is met. The holiday is logged in the Notes column with its date.

“Show union holidays in the notes column” is on by default. Uncheck it to hide the holiday names/dates from the notes everywhere — live view, list view, and the Excel export — without changing the schedule math.

Editing locks the country. Once you manually edit any comment, holiday, or hiatus band (text, color, or clearing one), the country selector locks — switching countries would recompute the holidays and discard your edits. Click Reset Notes & Hiatus above the calendar to restore the auto defaults, which unlocks the selector so you can switch countries again.

Simultaneous post

Check Auto-flag Simultaneous Post to mark weeks where post runs concurrently with the shoot. The offset field sets how many weeks after Production begins the flag starts. A flagged week shows a “Simultaneous Post” marker tucked into the leftmost free column right beside the running phase, so it takes as little space as possible.

Holiday hiatus

Add one or more hiatus periods (a start date and a number of weeks). A hiatus pushes whatever phase it falls inside outward so that phase still delivers its full requested week or day count — it doesn’t eat into the work. Hiatus weeks appear as a full-width “Holiday Hiatus” band. Click any hiatus band in Spreadsheet view to rename it or change its highlight color for that week.

Notes (in the preview)

The Notes column auto-fills key milestones inline — e.g. “Start Principal Photography 11/02/26”, “Principal Photography Wraps”, union holidays, and any simultaneous-post flags.

You can edit any note. In Spreadsheet view, click a note cell to open an editor. Type to replace it, or clear it completely to remove a pre-filled milestone — a cleared note stays gone and won’t regenerate. An empty note loses its highlight. The editor also shows the ten standard Microsoft Excel colors — pick one to change that individual cell’s highlight color (text flips black or white automatically for readability). Press Tab or Ctrl/Cmd‑Enter to save, Esc to cancel, or just click away. The Reset Notes & Hiatus button above the header returns every note, holiday, and hiatus band to its auto-generated text and default color — including bringing back any milestones you deleted.

Spreadsheet view vs. List view

Toggle at the top of the preview. Spreadsheet view mirrors the Excel export — a week-by-week grid with editable notes — and is the default. List view is a compact summary with each week’s phases and notes as badges.

Export to Excel

Generates a styled .xlsx matching the Spreadsheet view: one column block per year, colored phase cells, the milestone Notes column, and a bold printed header with the title, date, and production summary. Columns are sized tight to their content, and the whole grid is scaled to print as large as possible on a single sheet. Orientation is chosen automatically — calendars of two years or fewer print portrait, three or more years print landscape — unless the other orientation would fit noticeably larger. The file name uses your show title (e.g. “Show Title - Season 1 Planning Calendar.xlsx”).

Files, saving & reopening

All the main actions live in the toolbar at the top: the file menu (showing the calendar you’re currently in), New, Save, Save As, Export to Excel, and Reset.

Save writes a self-contained copy of this tool with all your entries baked in (title, dates, phases, custom phases, hiatuses, edited notes, header mode, and current view). In Chrome or Edge the first save asks where to put the file, and every save after that writes straight back to that same file with no dialog. Open the saved file later and everything comes back exactly as you left it.

New clears everything and starts a blank calendar. Save As is different — it copies the calendar you’re working on into a new file and switches you to it, which is handy for forking a schedule to try a variation without touching the original.

The file menu lists calendars you’ve saved; click one to switch to it, use Open… to pull in a calendar file that isn’t listed, or the × to drop a file from the list (that removes the shortcut only, never the file itself). Reopening a file may show a quick one-click permission prompt (“Let this site edit the file?”) — that’s the browser protecting your disk. The file menu and in-place saving need Chrome or Edge; in other browsers each save downloads a new copy instead.

Autosave & recovering work

The indicator beside Export shows Unsaved changes or the time of your last save. Once a calendar is linked to a file, it autosaves every 10 minutes in the background — no dialog, nothing to click.

Work that hasn’t been saved to a file yet can’t autosave (browsers won’t create a file without you choosing where it goes), so the app keeps a rolling backup in your browser’s local storage instead. If a session ends with unsaved changes, the next time you open the app it offers to recover that work. Closing the tab with unsaved changes also triggers your browser’s “leave site?” warning.

The backup lives only in this browser on this computer — it’s never uploaded anywhere, and clearing your browsing data removes it. Treat it as a safety net; the saved file on your disk is still the real copy.

Reset

Reset (bottom of the panel) clears all your entries and returns every setting to its default state. Two narrower resets live above the calendar: Reset Notes & Hiatus restores just the notes, holidays, and hiatus bands, and Header: Switch to Auto returns the header to live auto-fill.

Tip: any date you type snaps to the Monday of its week, and all durations are honored in full even when holidays or hiatuses interrupt them.
